About L-Sprout

The L-system is named after Astrid Lindenmayer, a Hungarian theoretical biologist
and botanist. Lindenmayer used L-systems to describe the behaviour of plant
cells and to model the growth processes of plant development.

L-systems have also been used to model the morphology of a variety of organisms
and can be used to generate self-similar fractals such as iterated function systems.

I wrote a JavaScript program to simulate this effect.
Use the button "Generate Srub Pattern" to create more layers of complexity
onto the shrub, rendering the nearly real-life effect of a growing plant. Out will
print generations of instructions as the plant gets more and more complex. Each
generation is built off the previous.

Because of the complexity of the organism as its instructions grow over each
generation, there are some limitations. Depending on the starting DNA, you can
generate between the 3rd-6th generation maximum. Also, there is a strip on
on the left side of the page that says L-System DNA Archive Contribution. In this,
you can submit your own discoveries to the archive where other people can find it.
Maybe someday, this platform can grow to have a library of DNA for this specific
L-System.


Intructions

The DNA is read by the program and is interpreted as instructions
Here is the list of rules that the computer reads. The program uses an algorithm
known as
turtle graphicsWikipedia Page
. I have provided a list of the commands needed to
command the turtle algorithm below:

Character Intruction
F grow upwards
+ grow rightwards
- grow leftwards
[ save position; push
] continue at save; pop


Final words, in order to restart the script, please refresh the page. In doing so, the
random plant will be regenerated, so likewise, the seeded plant
can also be regenerated. Thank you for taking part in this small social media-like
platform about sharing and looking at plant DNA/code!

The JS code can be found by doing the following while on a plant page:
CTRL+SHIFT+I > Sources > CTRL+P > sketch.js

You can also save an image of your own L-Sprout by
